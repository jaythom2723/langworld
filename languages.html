Langworld

A worldbuilding project

About

Languages are the backbone and main focus of Langworld, hence the project's name. There is a heavy preference towards conlanging in this project rather than general worldbuilding. However, both will take place consecutively whether or not curators realize it.

The languages created for this project are to be described as "EXTINCT" or "ALIVE" with a total number of speakers
